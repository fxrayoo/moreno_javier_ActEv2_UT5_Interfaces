ESTRUCTURA:
src:
- main: Aquí está el “Main” que es donde se ejecuta el programa e inicia el Login.
- model: Contiene las clases de datos (usuario, nota) preparadas para ser guardadas.
- view: Esta es la interfaz gráfica (login, menu).
- logica: Contiene la clase gestion_nota que maneja las operaciones de la lista.
- persistencia: Gestión de lectura y escritura de ficheros .dat.
- utils: Herramientas de seguridad para el encriptado de contraseñas.
------------------------------------------------------------------------------------------------------
COMO EJECUTAR EL PROYECTO
Clonar el repositorio.
Abrir en VS Code.
Asegurarse de tener el JDK instalado.
Ejecutar la clase “Main” ubicada en “src/main/Main.java”.
